# Extensão Kobra para VS Code

## Instalação

1. Instale as dependências:
```bash
npm install
```

2. Compile a extensão:
```bash
npm run compile
```

3. Instale a extensão:
```bash
code --install-extension kobra-language-0.1.0.vsix
```

## Uso

- Abra arquivos `.kobra` no VS Code
- Use `Ctrl+F5` para executar o arquivo
- Ou clique no botão "Play" na barra de título

## Funcionalidades

- Syntax highlighting para Kobra
- Execução de arquivos .kobra
- Auto-indentação
- Comentários com #
